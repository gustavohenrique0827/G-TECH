# GTech Technology — Site React + Vite

Este projeto foi migrado de Next.js para React usando Vite, mantendo o layout original e atualizando a identidade para laranja.

## Stack

- **React 18**
- **Vite**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React**

## Estrutura de pastas

```
gtech-site/
├── components/
├── public/
├── src/
│   ├── App.tsx
│   ├── globals.css
│   ├── main.tsx
│   └── vite-env.d.ts
├── tailwind.config.ts
├── tsconfig.json
├── vite.config.ts
├── index.html
├── package.json
└── postcss.config.js
```

## Como rodar localmente

```bash
npm install
npm run dev
```

Acesse http://localhost:5173

## Identidade visual

- **Cor primária:** laranja (`#F97316`)
- **Cor de destaque:** laranja claro (`#FB923C`)
- **Fundo:** azul quase preto (`#05070C`)
- **Tipografia:** fontes de sistema com fallback local
